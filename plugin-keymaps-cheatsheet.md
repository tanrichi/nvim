# **My Plugins Keymaps Cheatsheet**

## Completions:
* `<C-b>` scroll docs -4
* `<C-f>` scroll docs +4
* `<C-space>` complete
* `<C-e>` abort
* `<C-R>` confirm(select = true)
 
## LSP Configs:
* `K` buf.hover
* `<leader>gd` Go to definition
* `<leader>gr` Go to references
* `<leader>ca` List code actions
 
## Toggle:
* `<C-\>` Toggle terminal
 
## Telescope:
* `<leader>ff` Find files
* `<leader>fg` Grep
* `<leader>fb` Find buffers
* `<leader>fh` Help tags
* `<leader>fe` Find .env files
 
## None LS:
* `<leader>gf` Format
